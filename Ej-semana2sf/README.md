# Ejercicios de Desarrollo Fullstack - Semana 2

Este repositorio contiene una serie de ejercicios prácticos para aprender desarrollo web con HTML, CSS y JavaScript. Cada ejercicio incluye archivos HTML y JavaScript, junto con documentación detallada.

## Estructura del Proyecto

- `appX.js`: Archivo JavaScript con la lógica del ejercicio X
- `indexX.html`: Archivo HTML con la interfaz del ejercicio X
- `ejercicioX.md`: Documentación detallada del ejercicio X

## Lista de Ejercicios

### [Ejercicio 1: Conversor de Temperatura](ejercicio1.md)
- **Archivos**: [`app1.js`](app1.js), [`index1.html`](index1.html)
- **Descripción**: Conversor de grados Celsius a Fahrenheit con validación básica.

### [Ejercicio 2: Contador de Clics](ejercicio2.md)
- **Archivos**: [`app2.js`](app2.js), [`index2.html`](index2.html)
- **Descripción**: Contador simple con botones para incrementar y reiniciar.

### [Ejercicio 3: Cambiador de Color de Fondo](ejercicio3.md)
- **Archivos**: [`app3.js`](app3.js), [`index3.html`](index3.html)
- **Descripción**: Alterna entre modo claro y oscuro en el fondo de la página.

## Cómo Ejecutar los Ejercicios

1. Abre el archivo `indexX.html` correspondiente en un navegador web.
2. Interactúa con la interfaz según las instrucciones de cada ejercicio.
3. Consulta la documentación en `ejercicioX.md` para detalles técnicos.

## Tecnologías Utilizadas

- **HTML5**: Estructura y semántica de las páginas
- **CSS3**: Estilos y diseño responsivo
- **JavaScript (ES6+)**: Lógica interactiva y manipulación del DOM

## Notas

- Los ejercicios 4-12 no están implementados aún en este repositorio.
- Cada ejercicio está diseñado para ser autocontenido y educativo.
- Se recomienda completar los ejercicios en orden para un aprendizaje progresivo.
